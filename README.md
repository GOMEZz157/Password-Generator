# 🔐 Password Generator

Um gerador de senhas aleatórias desenvolvido utilizando **HTML, CSS e JavaScript**.
A aplicação permite gerar senhas seguras de forma rápida e simples, ajudando a aumentar a segurança de contas e sistemas.

---

# 🚀 Demonstração

Acesse o projeto online:

🔗 https://gomezz157.github.io/Password-Generator/

---

# ⚙️ Funcionalidades

* Gerar senhas aleatórias instantaneamente
* Escolher o **tamanho da senha**
* Incluir ou remover:

  * Letras maiúsculas
  * Letras minúsculas
  * Números
  * Símbolos
* Copiar a senha gerada com um clique
* Interface simples e fácil de usar

---

# 💻 Como executar o projeto

Abra o projeto live: [https://password-generator-three-orpin.vercel.app/](https://password-generator-three-orpin.vercel.app/)

ou

Clone o repositório:

```bash
git clone https://github.com/GOMEZz157/Password-Generator.git
```

Entre na pasta do projeto:

```bash
cd Password-Generator
```

Abra o arquivo `index.html` no navegador.
